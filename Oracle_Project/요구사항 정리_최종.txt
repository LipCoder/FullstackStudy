1. 권한 확인 함수
(프로시저를 실행하기 전에 권한을 먼저 확인한 후
프로시저를 수행할 지 결정한다)

2. 교수자 등록, 수정, 삭제 프로시저(관리자 권한)
(입력정보 : 교수자 이름, 뒷자리)
(기본 패스워드로 주민번호 뒷자리)

*추가사항->교수자 이름 컬럼 추가

3. 교수자 정보 출력 프로시저(관리자 권한)
(출력 정보 : 교수이름, 배정된 과목, 과목기간, 교재명, 강의실, 강의 진행 여부)

3-1. 강의 진행 여부 확인 함수

4. 성적 처리 프로시저(교수자 권한)

5. 과정 등록, 수정, 삭제 프로시저(관리자 권한)
(입력 정보 : 과정명, 과정기간, 강의실 정보)

6. 과정 정보 출력 프로시저(관리자 권한)
(출력 정보 : 과정명, 강의실, 과목명, 과목 기간, 교재명, 교수자 명)

7. 과목 등록, 수정, 삭제 프로시저(관리자 권한)
(입력 정보 : 과정명, 과목명, 과목기간, 교재명, 교수자 명)

8. 과목 정보 출력 프로시저(관리자 권한)
(출력 정보 : 과정명, 강의실, 과목명, 과목기간, 교재명, 교수자 명)

9. 학생 등록, 수정, 삭제 프로시저(관리자 권한)
   (입력 정보 : 학생 이름, 뒷자리)
   (기본 패스워드로 주민번호 뒷자리)

*추가사항->학생 이름 컬럼 추가

10. 학생 정보 출력 프로시저(관리자 권한)
(출력 정보 : 학생 이름, 과정명, 수강과목, 수강과목 총점)
(중도 탈락 여부 확인 가능)

11. 성적 입력, 수정, 삭제 프로시저(교수자 권한)
(자신의 과목을 수강하는 학생의 성적만 접근 가능)
(입력 정보 : 배점, 대상 학생, 출결점수, 실기점수, 필기점수)
(배점은 총합이 100점이여야 함)
(입력시 출결점수, 실기점수, 필기점수 합이 자동으로 화면에 출력)
(성적 입력시 일정 조건에 만족되지 못한 경우에는 중도 탈락자로 자동 등록된다.)

12. 성적 출력 프로시저(학생 권한)
(자신의 성적만 출력 가능)
(수강중인 모든 과목의 성적 확인 가능해야 함)
(중도 탈락자도 출력되어야 함, 단 탈락 여부 확인 가능해야됨)
(출력 정보 : 학생 이름, 과정명, 과목명, 교육 기간, 교재명, 점수(출결, 실기, 필기), 총점, 등수)

13. 배점 입력, 수정, 삭제 프로시저(교수자 권한)

14. 중도 탈락자 입력, 수정, 삭제 프로시저(관리자 권한)
