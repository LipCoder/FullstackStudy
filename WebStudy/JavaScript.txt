Java Script의 특징

1. 동적 타이핑 (Dynamic Typing)
var을 사용한다. 정적 타이핑과 다르게 자료형을 컴파일 시점이 아닌 런타임 시 결정한다.

2. 객체 지향 언어, 프로그래밍 (Object Oriented Programming)
Prototype이라는 형식이 존재 → Class 처럼 사용
   
3. Client Side 언어 

4. 하지만 Server Side에서 JS를 사용할 수 있게 만든것 → Node.js

5. JQuery

6. JSON

7. 모든 공백 문자를 무시한다.




//
DOM(Document Object Model 의 약자)

 - 문서에 대한 모든 내용을 담고있는 객체. 도큐먼트에 관련 된 내용 모두

 - 문서 즉 열려진 페이지에 대한 정보를 따로 객체화 시켜서 관리함



BOM(Browser Object Model 의 약자)

 - 브라우저에 대한 모든 내용을 담고있는 객체. 브라우저에 관련 된 내용 모두

 - 뒤로가기, 북마크, 즐겨찾기, 히스토리, URL정보 등..

 - 브라우저가 가지고 있는 정보를 따로 객체화 시켜서 관리함
//
