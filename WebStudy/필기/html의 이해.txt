#1. HTML 해석기 - 웹 브라우저

- 웹 문서는 HTML(HyperText Markup Language)언어로 작성된 문서로 htm, html확장자로 저장된다.
- 우리는 웹 브라우저를 통해 HTML 언어가 번역된 결과 화면을 본다.
- HTML의 여러 명령어를 이용해서 웹 브라우저에 문자와 그림을 표현하거나 하이퍼링크로 연결한다. 
- HTML의 하이퍼링크를 통해서 전 세계 문서가 연결되기 때문에
  인터넷을 월드와이드웹(WWW)이라고 부른다.

웹문서(.htm, .html) -> 웹 브라우저에 의해 번역 -> 결과 화면 출력

#3. HTML 태그

*HTML 기본 구조

<HTML> 
	<HEAD>
		<TITLE> 웹 문서의 제목 </TITLE>
	</HEAD>
	<BODY>
		웹 문서의 내용(글자나 그림 등)
	</BODY>
</HTML>

1. HTML 문서는 <HTML> 로 시작해서 </HTML>로 끝난다. <HEAD>영역 <BODY> 영역이 있다.

2. TITLE 태그는 생략 가능


* 태그의 특징

1. 태그 이름은 대소문자를 구분하지 않지만 소문자로 작성하는 것을 권장한다.

2. 시작 태그 <>, 끝 태그 </> (한 쌍으로 사용되는 경우)
   <HR> (단독으로 쓰이는 태그)

3. 시각적인 부분은 HTML에서 제공하는 태그로 사용하면 안된다. 이건 CSS를 사용해야 한다. 범위를 침범해서는 안된다. (웹 표준의 기본 개념)

4. 태그는 중첩해서 사용할 수 있는데 반드시 시작과 끝을 맞춰야 한다. 가장 먼저 시작된 태그가 가장 나중에 끝이 난다는 원리이다.
(<a> <b> </b> </a> -> 올바름)
(<a> <b> </a> </b> -> 잘못됨)

# 하이퍼링크
 텍스트나 그림에 필요한 정보를 다양한 형태로 연결시킬 수 있다. 또한 일반 글자뿐만 아니라 각종 멀티미디어 요소들도 링크할 수 있다.
(<A> 태그와 속성을 같이 이용해야 한다. 속성 : href, name, target)
하이퍼링크를 설정할 때에 경로를 잘못 지정하게 되면 문서가 나타나지 않게 된다.
(400 번때 에러는 클라이언트 에러, 500 번때 에러는 서버 에러)
